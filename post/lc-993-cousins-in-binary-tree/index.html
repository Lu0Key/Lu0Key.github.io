993.二叉树的堂兄弟节点

在二叉树中, 根节点位于深度 0 处，每个深度为 k 的节点的子节点位于深度 k+1 处。

如果二叉树的两个节点深度相同，但 父节点不同 ，则它们是一对堂兄弟节点。

我们给出了具有唯一值的二叉树的根节点 root ，以及树中两个不同节点的值 x 和 y 。

只有与值 x 和 y 对应的节点是堂兄弟节点时，才返回 true 。否则，返回 false。

示例 1:

输入: root = [1,2,3,4], x = 4, y = 3
输出: false

示例 2:

输入: root = [1,2,3,null,4,null,5], x = 5, y = 4
输出: true

示例 3:

输入: root = [1,2,3,null,4], x = 2, y = 3
输出: false

提示：

  • 二叉树的节点数介于 2 到 100 之间。
  • 每个节点的值都是唯一的、范围为 1 到 100 的整数。

解答

本题的难度是 Easy.

思路很简单啊, 就是第一遍先存下每一个节点的父节点, 然后再存下每一个节点的祖父节点. 然后开始遍历, 如果两个节点的父节点相同, 那么就不是堂兄弟节点, 如果两个节点的祖(祖祖祖)父节点相同, 那么就是堂兄弟节点. 这里是要求深度相同的, 因此用一个 while 循环就能解决, 并且因为每个节点的值都是唯一的, 因此可以用数组来存储.

代码如下:

class Solution {
    private int[] parent = new int[101];
    private int[] grandparent = new int[101];
    public boolean isCousins(TreeNode root, int x, int y) {
        BTS(root);
        if(parent[x] == parent[y]){return false;}
        while(grandparent[x]!=0 && grandparent[y]!=0){
            if(grandparent[x] == grandparent[y]){return true;}
            x = parent[x];
            y = parent[y];
            // if(grandparent[x] == grandparent[y]){return true;}
        }
        return  false;
    }

    private void BTS(TreeNode node){
        if(node == null){return ;}
        if(node.left!=null){
            parent[node.left.val] = node.val;

            if(node.left.left!=null){
                grandparent[node.left.left.val] = node.val;
            }
            if(node.left.right!=null){
                grandparent[node.left.right.val] = node.val;
            }
            BTS(node.left);
        }

        if(node.right!=null){
            parent[node.right.val] = node.val;

            if(node.right.left!=null){
                grandparent[node.right.left.val] = node.val;
            }
            if(node.right.right!=null){
                grandparent[node.right.right.val] = node.val;
            }
            BTS(node.right);
        }
    }
}

因为节点数少, 复杂度又低, 因此时间开销为 0ms, 击败100% 的提交.
