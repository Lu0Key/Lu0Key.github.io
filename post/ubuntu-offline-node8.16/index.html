Ubuntu 离线安装 Node8.16

Nov 6, 2024
1 min read
Linux

1. 下载源码

进入官网选择所需要的版本, 下载, 这里我选择 node-v8.16.0-linux-x64.tar.gz

2. 解压

# 解压
tar -xvf node-v8.16.0-linux-x64.tar.gz -C ~/nodejs/

3. 配置软链接

# 配置软链接
sudo ln -s ~/nodejs/node-v8.16.0-linux-x64/bin/node /usr/local/bin
sudo ln -s ~/nodejs/node-v8.16.0-linux-x64/bin/npm /usr/local/bin
