Hello world

第一篇Hugo博客

下载主题

git clone https://github.com/lxndrblz/anatole.git themes/anatole

hugo 本地启动

hugo server -t themename --bulidDrafts

hugo 创建新文章

hugo new post/blogname.md

推到Github远端服务器

创建名为 [github用户名].github.io 的仓库

blog目录下

hugo --theme=anatole --baseUrl="https://Lu0key.github.io/" --buildDrafts

会生成一个public文件夹，进入该文件夹

cd public

初始化git仓库

git init

将所有文件添加入暂存区

git add .

将git仓库和云端仓库关联

git remote add [shortname] [url]

比如我的就是

git remote add origin https://github.com/Lu0key/Lu0key.github.io.git

最后推至云端

git push -u origin master

之后修改博客后，再推到GitHub上需要

hugo --theme=m10c --baseUrl="https://Lu0key.github.io/" --buildDrafts
cd public
git add .
git commit -m "message"
git push
