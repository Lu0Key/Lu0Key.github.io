网速

有时候想要看看当前服务器的上传下载速度，当前的流量怎么样, 因此需要

sudo apt-get update
sudo apt-get install vnstat
sudo vnstat -l


# 上面那个比较简陋
# 我们可以使用有可视化界面的
sudo apt-get update
sudo apt-get install bmon
sudo bmon

硬件

# 有时候要看没装显卡的服务器的型号是什么
lspci | grep VGA
# 16:00.0 VGA compatible controller: NVIDIA Corporation Device 2684 (rev a1)
# 34:00.0 VGA compatible controller: NVIDIA Corporation Device 2684 (rev a1)
# 52:00.0 VGA compatible controller: NVIDIA Corporation Device 2684 (rev a1)
# 70:00.0 VGA compatible controller: NVIDIA Corporation Device 2684 (rev a1)
# 然后就可以根据这个 2684 去下载显卡驱动
# https://admin.pci-ids.ucw.cz/mods/PC/10de?action=help?help=pci
# 通过这个网站看显卡的型号

# 有时候可能会好奇主板型号
sudo dmidecode -t baseboard
# 可以用这个命令查看
# Manufacturer 厂家
# Product Name 型号

# 查看内存信息
sudo dmidecode -t memory
# 可以看 Manufacturer 知道厂家是哪个

# 查看显卡厂家
lspci -vnn | grep VGA -A 12

# 查看硬盘的型号
lsblk -o NAME,MODEL
