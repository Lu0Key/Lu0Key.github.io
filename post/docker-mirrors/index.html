Docker 镜像下载加速

Nov 19, 2024
1 min read
Linux

当前，Docker 镜像下载速度较慢，可以通过配置 Docker 镜像加速器来提高下载速度。

sudo vim /etc/docker/daemon.json

修改里面的内容为

{
        "registry-mirrors": [
                                "https://yczv4b52.mirror.aliyuncs.com",
                                "https://docker.m.daocloud.io",
                                "https://huecker.io",
                                "https://dockerhub.timeweb.cloud",
                                "https://registry.cn-hangzhou.aliyuncs.com"
        ],
        "live-restore": true,
        "log-opts": {
                "max-size": "512m",
                "max-file": "3"
        },    
        "dns": ["127.0.0.53", "8.8.8.8", "114.114.114.114"]
}

主要是这里的 registry-mirrors 配置项，可以配置多个镜像地址，以逗号分隔。

然后重启docker

sudo systemctl daemon-reload
sudo systemctl restart docker

这样就可以加速 Docker 镜像的下载速度了。
